# Marketing ideas

## Create website: chatbot.torvian.eu
- create landing page with basic info about the project
- create page with documentation for developers (API reference, etc.)
- enable users to try out Torvian chatbot live on the website 
  - host chatbot server on: https://chatbot-server-demo.torvian.eu. Rent VPS on OVHcloud.
  - pre-configure admin account with public access to free LLMs from OpenRouter
  - when user clicks "Try it out" button on the website, create a new user account for them and connect them to the chatbot server.
  - host WASM chatbot client on: https://chatbot-client-demo.torvian.eu
  - (optional) implement remote MCP server feature, because local MCP servers are not supported in browser
